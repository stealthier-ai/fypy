==============================
fypy使用说明
==============================

.. note::

    由于团队成员减少，功能开发、测试、发布受到影响，
    将争取每个月发布一版，增加遥感数据处理、气象计算公式、
    栅格矢量基础操作等功能。

    未来计划：

    - 风云卫星产品数据的处理、图像绘制等功能
    - 气象行业常用的计算公式
    - 常用的散点图、直方图、时序图绘制等

.. warning::

    当前 ``fypy`` 库还不是一个稳定的版本，正在对接口API
    进行测试补充，争取把接口满足大家需求。



``fypy`` 是一个支持 ``python`` 的库，并支持以下功能：


+------------------------+----------------------+--------------------------------+----------+
|         库名           |    对象              |         功能                   |   备注   |
+========================+======================+================================+==========+
| fypy                   | fy3                  | 极轨卫星FY3的5分钟块、轨道数据 |          |
|                        |                      | 的投影、拼接裁剪               |          |
|                        +----------------------+--------------------------------+----------+
|                        | fy4                  | 卫星数据的经纬度转行列号，     |          |
|                        |                      | 行列号转经纬度，以及对数据进行 |          |
|                        |                      | 投影转换、裁剪等功能（待发布） |          |
|                        +----------------------+--------------------------------+----------+
|                        | h8                   |                                |          |
|                        +----------------------+--------------------------------+----------+
|                        | modis                |                                |          |
+------------------------+----------------------+--------------------------------+----------+
